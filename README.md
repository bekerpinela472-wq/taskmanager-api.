# taskmanager-api.
Programa de Suma de Dos Números
Version 1.0.0

Descripción
Este proyecto consiste en un programa básico desarrollado en Python que permite sumar dos números ingresados por el usuario. Es un ejercicio introductorio para comprender conceptos fundamentales de programación, como el uso de variables, entrada de datos mediante teclado, operaciones matemáticas y salida de información en pantalla.

Características
Solicita al usuario dos números.
Realiza la suma de ambos valores.
Muestra el resultado en pantalla.
Utiliza el tipo de dato float para permitir números enteros y decimales.
Requisitos
Python 3.x instalado en el sistema.
Instalación
Descarga o copia el código fuente.
Guarda el archivo con el nombre suma.py.
Verifica que Python esté instalado ejecutando:
python --version
o

python3 --version
Uso
Ejecuta el programa desde la terminal:

python suma.py
o

python3 suma.py
Código del programa
# Programa para sumar dos números

numero1 = float(input("Ingrese el primer número: "))
numero2 = float(input("Ingrese el segundo número: "))

suma = numero1 + numero2

print("La suma es:", suma)
Ejemplo de ejecución
Ingrese el primer número: 10
Ingrese el segundo número: 5
La suma es: 15.0
